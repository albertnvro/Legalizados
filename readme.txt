== Descripción ==
Legal Pages Generator es un plugin que te permite generar automáticamente las páginas de Aviso Legal y Política de Privacidad en tu sitio de WordPress, cumpliendo con las normativas legales en España, incluyendo el RGPD.

== Características ==
* Generación automática de las páginas de Aviso Legal y Política de Privacidad.
* Cumplimiento con las normativas legales en España.
* Fácil de configurar y personalizar.
* Estilo de formulario limpio y moderno.

== Instalación ==
1. Sube la carpeta del plugin al directorio `/wp-content/plugins/` o instala el plugin directamente desde WordPress.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Accede a 'Legal Pages Generator' en el menú de administración de WordPress para configurar el plugin.
4. Rellena el formulario con la información requerida y haz clic en 'Crear Páginas' para generar las páginas automáticamente.

== Preguntas Frecuentes ==

= ¿El plugin cumple con el RGPD? =
Sí, el plugin está diseñado para cumplir con las normativas del RGPD en España, asegurando que las páginas generadas contienen la información necesaria para estar en conformidad con la ley.

= ¿Puedo personalizar las plantillas generadas? =
Sí, puedes editar las plantillas `aviso-legal-template.php` y `politica-privacidad-template.php` en la carpeta del plugin para adaptarlas a tus necesidades específicas.
